# BMI Calculator for CloudPhone

A simple and modern BMI (Body Mass Index) calculator app designed for CloudPhone devices (240×320 resolution).  
Features include:
- **Metric & Imperial Units**: Switch between cm ↔ ft and kg ↔ lbs.  
- **Material Design UI**: Clean layout with responsive top/bottom bar.

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/BMI/index.html`  
4. Upload icons if required.  
5. Save and refresh — **BMI Calculator** will appear on your CloudPhone.
